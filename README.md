# Calibration System

A web-based calibration management system built with PHP and MySQL.

## Features
- Equipment calibration tracking
- Audit logging
- User management

## Tech Stack
- PHP
- MySQL (via XAMPP)
- HTML/CSS/JavaScript

## How to Run
1. Install XAMPP
2. Clone this repository into your `htdocs` folder
3. Import the database
4. Open `localhost/calibration_system` in your browser
